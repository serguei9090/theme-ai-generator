# Documentation Directory

This directory contains the core documentation for the Theme AI Generator project, including architecture, SDK integrations, and submission materials.

## 📁 Folders

- **[CopilotSDK](./CopilotSDK)**: Detailed technical guides and implementation notes for the GitHub Copilot SDK integration.
  - **[nodejs](./CopilotSDK/nodejs)**: Specific recipes and patterns for Node.js environments (error handling, session persistence, etc.).
- **[screenshots](./screenshots)**: Visual gallery of the application interface, settings, and generated themes used for submissions.

## 📄 Key Documentation Files

- **[architecture.md](./architecture.md)**: Comprehensive overview of the system design, including the frontend/backend contract, MCP server integration, and palette engine logic.
- **[copilotsdk.md](./copilotsdk.md)**: Primary documentation for the project's use of the GitHub Copilot SDK for theme generation.
- **[PROMPT_HIGHLIGHTS.md](./PROMPT_HIGHLIGHTS.md)**: A collection of high-performing prompts and "director" intent interpretations used to train and refine the AI's behavior.
- **[SUBMISSION_TIPS.md](./SUBMISSION_TIPS.md)**: Guidelines and best practices for preparing the final project submission.
- **[eventReequirement.md](./eventReequirement.md)**: Tracking and checklists for meeting the Agents League submission requirements.
- **[eventcompilant.md](./eventcompilant.md)**: Compliance verification notes for the target submission event.
- **[copilot_prompt_respopnse-RAW.md](./copilot_prompt_respopnse-RAW.md)**: Raw logs and examples of LLM interactions during the development of the Copilot integration.
